# @codygo-ai/vite-tools

Vite utilities and tools for CodyGo AI projects.

## Installation

```bash
npm install --save-dev @codygo-ai/vite-tools
```

## Requirements

- Vite 5.x or 6.x

## Usage

```typescript
import { /* utilities */ } from '@codygo-ai/vite-tools';
```

## Development

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```

### Type Check

```bash
npm run typecheck
```
